  <h1>Drone Simulation System (C++)</h1>

  <h2>Description</h2>
  <p>
    Developed a drone simulation system <b>(C++)</b> that manages drone movement, environment interaction, and timed mine behavior.<br>
    The project is built using object-oriented design and SOLID principles, ensuring clean, maintainable, and scalable code.<br>
    Applied design patterns to structure the system in a modular and extensible way, including a command-based approach for handling actions.<br>
    Incorporated basic multithreading to support time-based events such as automatic mine expiration.<br>
    Ensured safe access to shared resources using synchronization mechanisms.
  </p>

  <h2>Utilities Used</h2>
  <ul>
    <li>Object-Oriented Programming (OOP)</li>
    <li>SOLID principles</li>
    <li>Design patterns</li>
    <li>Basic multithreading</li>
    <li>Mutex synchronization</li>
